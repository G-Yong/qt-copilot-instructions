# Qt Copilot Instructions

使能 GitHub Copilot 创建、编译、运行和调试 Qt 工程的自定义指令集。

## 功能特性

- 🔧 自动识别 qmake 和 CMake 项目
- 🏗️ 指导项目配置与编译流程
- 🐛 提供调试支持和故障排除
- 📦 检查 Qt 环境和依赖

## 使用方法

将本仓库克隆到工作区，GitHub Copilot 将自动加载 `.github/copilot-instructions.md` 中定义的 Qt 开发 Skill。

## 环境要求

- Qt 5.15+ (默认配置为 Qt 5.15.2 MSVC2019 64-bit)
- qmake 或 CMake
- Windows 环境 (可根据需求调整)
